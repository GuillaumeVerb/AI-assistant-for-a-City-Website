# AI Assistant for a City Website - V0

## Goal
Build a simple AI assistant that can answer basic citizen questions and later route them to the right city service.

## Week 1 scope
- call an LLM from Python
- support mock and real modes
- save outputs as JSON
- prepare a clean project structure

## Current status
The project can:
- send questions to a mock or real LLM
- receive answers
- save answers locally

## Next step
Add structured outputs and category classification.
